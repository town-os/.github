# Town OS

## Your Cloud in Your Closet, easy enough for anyone.

**[https://town-os.github.io](https://town-os.github.io)**

Town OS is a self-service platform that anyone can run at home. It lives on a USB drive, runs entirely in memory, and manages its own storage, services, and networking. Upgrade by replacing a USB drive. Reset by rebooting.

Source: [gitea.com/town-os/town-os](https://gitea.com/town-os/town-os)
